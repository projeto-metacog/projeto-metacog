Tecnologias Utilizadas

Front-End

O front-end da aplicação é construído utilizando React, uma biblioteca JavaScript para interfaces de usuário, em conjunto com o framework Next.js, que traz funcionalidades como renderização no lado do servidor e geração de páginas estáticas. Para a estilização, utiliza-se Styled Components, permitindo a escrita de CSS diretamente dentro dos componentes.

Back-End

A lógica do back-end é provisionada pelo próprio Next.js, que oferece suporte a APIs do lado do servidor. Com isso, é possível implementar a comunicação entre o cliente e o servidor de forma eficiente e integrada.

Banco de Dados

A aplicação utiliza o PostgreSQL como banco de dados relacional. Para facilitar a interação com o banco de dados, é empregado o ORM Prisma, que permite criar e executar consultas complexas de forma simples e intuitiva.

Infraestrutura

O Docker é usado para conteinerizar a aplicação, garantindo consistência entre os ambientes de desenvolvimento e produção. Isso facilita o gerenciamento do ambiente e a instalação de dependências.
