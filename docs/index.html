Sobre o projeto

O projeto Metacognição em Ação: Explorando o Mundo da Leitura, vinculado à Universidade Federal de São Paulo (Unifesp), tem como objetivo desenvolver uma ferramenta digital interativa para o aprimoramento da metacognição em leitura. A iniciativa busca melhorar capacidades como dedução, indução, raciocínio lógico, pensamento crítico e entendimento de textos, especialmente voltadas para crianças do ensino fundamental.

Seja você um desenvolvedor, um usuário ou apenas alguém interessado, aqui você encontrará todas as informações necessárias sobre o projeto.

Introdução

  • Passo 1: Clone o repositório: git clone https://github.com/henriqueodaguiri/PEC-Metacom
  • Passo 2: Instale as dependências: npm install
  • Passo 3: Execute a aplicação: npm start

Features

  • Aprimoramento de dedução e indução em leitura.
  • Exercícios interativos para desenvolver raciocínio lógico.
  • Feedback instantâneo para aprimorar o pensamento crítico.
